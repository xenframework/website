Download xenframework

  1. You want to create a new project
  2. You only need the latest version of xenframework
  3. You need to update your xenframework installation
  4. You want to fork or clone the repositories on GitHub

Composer

In the first 3 cases you will need composer. If you do not have composer already installed run this: 

curl -s https://getcomposer.org/installer | php --

more detailed instructions here: https://getcomposer.org/doc/00-intro.md

1. Creating a new project

This will create a new project with xenframework and the others dependencies already installed.

Install the skeleton-app using composer 

php composer.phar create-project -sdev xenframework/skeleton-app myproject

If you did a global install and do not have the phar in that directory run this instead: 

composer create-project -sdev xenframework/skeleton-app myproject

more detailed instructions here: https://github.com/xenframework/skeleton-app/blob/master/README.md

2. Getting the latest version of the xen library

Create a composer.json file with this inside: 

{
    "require": {
        "xenframework/xen": "2.0.0.*@dev"
    }
}

Now run composer install 

php composer.phar install

or if you did a global composer installation 

composer install

3. Updating xenframework

You can update single packages. If you only need to update xenframework library located in your vendor directory: 

composer.phar update xenframework/xen

more info: https://getcomposer.org/doc/03-cli.md#update

4. Github

Alternatively you can download, clone or fork the repositories from GitHub: 

https://github.com/xenframework

Take care about each package dependencies (look at the requires section):

  • skeleton-app - https://packagist.org/packages/xenframework/skeleton-app
  • xenframework/xen - https://packagist.org/packages/xenframework/xen
